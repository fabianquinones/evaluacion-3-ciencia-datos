# Proyecto End-to-End de Ciencia de Datos - CarSeats

Este repositorio contiene un proyecto end-to-end de ciencia de datos aplicado al dataset CarSeats.

## Objetivo

Analizar los datos de ventas de tiendas, preparar la información, entrenar un modelo predictivo con Random Forest y presentar resultados mediante visualizaciones y dashboard interactivo.

## Estructura del proyecto

- data/raw/: dataset original.
- data/processed/: datos procesados generados durante el análisis.
- 
otebooks/: notebook principal del proyecto.
- src/: scripts auxiliares del proyecto.
- dashboard/: archivos relacionados con el dashboard en Dash.
- docs/: documentación del proyecto.
- eports/: reportes o resultados finales.
- ssets/: imágenes o recursos visuales.

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Plotly
- Dash
- Random Forest

## Modelo utilizado

Se utiliza Random Forest por su capacidad para trabajar con variables numéricas y categóricas codificadas, capturar relaciones no lineales y entregar un buen rendimiento sin requerir supuestos estadísticos estrictos.
